Website options

Specify keywords to enable the extension to run on any sites with URLs containing these keywords.
If none specified, extension will not run (You can't run it on all pages, that's just going to be annoying).
You can use wildcards(like *) as well.
You can also specify multiple websites separated by a comma(,).

Examples: linkedin*  facebook,google glassdoor
